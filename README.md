#### O cálculo do Imposto sobre Circulação de Mercadorias e Serviços (ICMS)
### 1. Entenda a Base de Cálculo e a Alíquota
```
A base de cálculo do ICMS é, geralmente, o valor total da operação (preço do produto ou serviço, 
frete, seguro e outras despesas acessórias). A alíquota varia de acordo com o estado e o tipo de 
produto.
```
### 2. A Fórmula do Cálculo "Por Dentro"
```
No Brasil, o ICMS é um imposto que incide "por dentro", ou seja, o seu valor já está embutido 
na própria base de cálculo. A fórmula correta para encontrar a base de cálculo (BC) é:
```
## Base de Cálculo (BC) = Valor da Operação / (1 – Alíquota do ICMS)
# Após encontrar a BC, você pode calcular o valor do ICMS devido:
# Valor do ICMS = Base de Cálculo (BC) x Alíquota do ICMS</p>

### 3. Exemplo Prático
```
Imagine uma venda de produto no valor de R$ 1.000,00 dentro do estado de São Paulo, 
onde a alíquota interna é de 18% (0,18).
```

#### Passo 1: Calcular a Base de Cálculo (BC) ajustada
* BC = R$ 1.000,00 / (1 – 0,18)
* BC = R$ 1.000,00 / 0,82
* BC ≈ R$ 1.219,51

#### Passo 2: Calcular o Valor do ICMS
* Valor do ICMS = R$ 1.219,51 x 0,18
* Valor do ICMS ≈ R$ 219,51
#### Nesse caso, o valor do ICMS a ser destacado na nota fiscal é de aproximadamente 
#### R 219,51, e o valor total da operação é R 1.219,51.
